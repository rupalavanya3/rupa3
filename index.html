Coming to the topic,HTML is the language in which most websites are written.HTML is used to create pages and make them functional.

The code used to make them visually appealing is known as CSS and we shall focus on this in a later tutorial.

The HTML stands for Hyper Text Markup Language.Here hyper text means that the document contains links that allow the reader to jump to other places in the document or to another document altogether.The latest version is known as HTML5.

A Markup Language is a way that computers speak to each other to control how text is processed and presented.To do this HTML uses two things-tags and attributes.Tags and attributes are the basis of HTML.

TAGS -->used to mark up the start of an HTML element and they are usually enclosed in angle brackets.An example of a tag is heading tags h1,h2,h3,h4,h5,h6 enclosing angle brackets
Ex:

<h1>...heading...</h1>

ATTRIBUTES-->they contain additional pieces of information.Attributes take the form of an opening tag and additional info is placed inside.
Ex:


<img src="mypic.jpg" alt="A photo of mine.">

In this instance, the image source (src) and the alt text(alt) are attributes of the <img>

NOTE:When using multiple tags,the tags must be closed in the order in which they were opened.
Ex:

<strong><em>This is really important!</em></strong>

Links are included in attribute opened by the <a> tag.This element is the first that we've met which uses an attribute and so it looks different to previously mentioned tags.

The Anchor Tag

This tag is used to hyperlink the text,file,images taht we see in website.It contains mainy attributes like 'href' which is used to hyperlink that website and it just open it, 'ping' which is used to make trackers, 'download' when clicked just simply downloads that content. The <a>(or anchor) opening tag is written in the format:

<a href="https://google.com">Google

The first part of the attribute points to the page that will open once the link is clicked.Meanwhile ,the second part of the attribute contains is the text which will be displayed to a visitor in order to entice them to click on the link.

Adding images in HTML

In Today's modern digital world,images are everything.The <img> tag has everything you need to display images in site.Like anchor tag <img> also contains an attribute.

The attribute features info for your computer regarding the source,height,width and alt text of the image.

Alt text is important to ensure that your site is ranked correctly on search sites and also for visually impaired visitors to your site.

When drawing a table we must open an element with the <table> opening tag.Inside this, we structure the table using the table rows,<tr> and cells,<td>
Ex:

<table>
<tr>
<td>Row 1-Coloumn 1</td>
<td>Row 1-Coloumn 2</td>
<td>Row 1-Coloumn 2</td>
</tr>
<tr>
<td>Row 2-Coloumn 1</td>
<td>Row 2-Coloumn 2</td>
<td>Row 2-Coloumn 3</td>
</tr>
</table>

This will produce a 2-row table with 3 cells in each row.
